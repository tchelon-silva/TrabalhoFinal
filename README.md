## 3. Descrição

Este é um projeto de grupo da UC00617. Consiste num pequeno site em HTML, desenvolvido com o objetivo principal de praticar Git e GitHub: criação de branches, commits, abertura e revisão de Pull Requests, e resolução de conflitos.

## 5. Páginas

- `index.html` — página principal do projeto, com a apresentação da equipa, a forma de trabalho e o objetivo do projeto.
- `membro-tiago.html` — página pessoal de Tiago Moega.
- `membro-gabriel.html` — página pessoal de Gabriel Santos.
- `membro-tchelon.html` — página pessoal de Tchelon Silva.

## 6. Trabalho realizado

- Criação das páginas pessoais de cada membro da equipa.
- Construção conjunta do `index.html`.
- Uso de branches para trabalhar de forma isolada.
- Uso de commits pequenos e frequentes.
- Abertura e revisão de Pull Requests entre colegas.
- Resolução do conflito obrigatório na linha do lema.

## 7. Tecnologias usadas

- HTML
- CSS fornecido pelo formador
- Git
- GitHub
- GitHub Pages

## 8. Organização do trabalho

- Cada página pessoal foi desenvolvida num branch próprio.
- O `index.html` foi construído em várias sessões de trabalho.
- Todos os Pull Requests foram revistos por um colega antes do merge.
- Ninguém fez merge do seu próprio Pull Request.
- O conflito obrigatório (na linha do lema) foi identificado e resolvido antes da entrega final.

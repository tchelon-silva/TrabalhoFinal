O que estou a aprender

Estou a aprender a utilizar os serviços git e github.

  • Branches: São ramos do ficheiro principal (main) onde podem adicionar novas funcionalidades (features).
  • Commits: é a etapa onde atualizamos nosso repositório local
  • Pull Requests: É uma solicitação para autorizar as mudanças de uma branch para a main principal

Ferramentas usadas

Frase sobre como usaste ferramentas no projeto.

  • Git: usado como repositório local.
  • GitHub: usado como repositório remoto.
  • VS Code: usado para criar linhas de código e executar comandos no terminal.

Comandos praticados

Frase sobre o que praticaste durante o trabalho.

  • Criar commits: esse comando é feito para atualizar nosso repositório local.
  • Criar branches: esse comando é feito para adicionar novas funcionalidades a partir da branch principal main.
  • Enviar um branch para o GitHub: esse comando é feito com git push e depois pode ser feito uma solicitação para a branch se integrar na main usando Pull Request

Início

Sobre mim

Chamo-me Gabriel Santos e estou a frequentar este curso para reforçar as minhas competências em tecnologia.

Gosto de resolver problemas passo a passo e de entender como as coisas funcionam por dentro, em vez de me ficar só pelo resultado final.

Este projeto é, para mim, a primeira vez que trabalho com Git e GitHub num contexto de equipa.

Percurso

Antes deste curso, já tinha algum contacto com computadores no dia a dia, mas nunca tinha usado Git ou GitHub de forma organizada.

A parte que tem sido mais desafiante é perceber bem o que acontece em cada comando antes de o executar, mas é também a que mais me motiva a continuar.

O que estou a aprender

Ao longo desta unidade tenho vindo a perceber melhor como funciona o trabalho em equipa num projeto de programação.

  • Trabalhar em branches separados sem afetar o trabalho dos colegas
  • Fazer commits pequenos, com mensagens claras sobre o que mudou
  • Abrir e rever Pull Requests antes de juntar código ao main

Comandos praticados

Durante a criação desta página, pratiquei vários comandos e ações que se repetem no dia a dia com Git.

  • Criar um branch novo para a minha página
  • Verificar alterações com git status e git diff antes de cada commit
  • Enviar o branch para o GitHub e abrir um Pull Request
